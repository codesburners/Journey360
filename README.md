Journey360
AI-Powered Smart Travel Companion

Journey360 is a mobile-first app that uses AI to plan trips, provide real-time assistance, ensure safety, and generate post-trip insights. Built in 10 days for a college project.

Features
Smart Itinerary Builder: AI-generated personalized travel plans.
AI Travel Assistant: Context-aware chatbot for trip questions.
Safety Dashboard: Real-time risk monitoring with SOS.
Post-Trip Summary: AI insights and reflections.
Tech Stack
Frontend: React, Tailwind CSS
Backend: FastAPI, MongoDB
AI: OpenAI API, Google Maps, NewsAPI
Installation
Clone the repo: git clone https://github.com/codesburners/Journey360.git
Backend: cd backend && pip install -r requirements.txt && uvicorn main:app --reload
Frontend: cd frontend && npm install && npm run dev
Add API keys to .env.
Usage
Sign up/login with Firebase.
Create a trip and generate an itinerary.
Use the assistant, monitor safety, and view summaries.
